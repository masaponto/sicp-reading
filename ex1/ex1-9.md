# 1 
```lisp
(define (+ a b)
    (if (= a 0) b (inc (+ (dec a) b))))
```
```
(+ 3 5)
(inc (+ 2 5))
(inc (inc (+ 1 5)))
(inc (inc (inc (+ 0 5))))
(inc (inc (inc 5)))
(inc (inc 6))
(inc 7)
8
```
線形再帰プロセス


# 2
```lisp
(define (+ a b)
    (if (= a 0) b (+ (dec a) (inc b))))
```

```

(+ 2 6)
(+ 1 7)
(+ 0 8)

```
